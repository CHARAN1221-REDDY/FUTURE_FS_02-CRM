# Advanced Mini CRM - Client Lead Management System

## Overview

Advanced Mini CRM is a full stack Client Lead Management System designed to help businesses, freelancers, and startups manage client leads efficiently.

This project allows users to:
- Add and manage leads
- Update lead status
- Delete leads
- Search leads
- View dashboard analytics
- Store data securely using MongoDB

The application simulates a real-world CRM workflow used by agencies and businesses for customer management and follow-ups.
